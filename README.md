# Ранобэ Архив

Личный архив русскоязычных ранобэ в формате epub.

## Структура файлов

```
ranobe-archive/
├── index.html        ← главная страница (публичная)
├── admin.html        ← админка для добавления контента
├── data.json         ← база данных всех ранобэ
├── covers/           ← обложки книг (jpg/png)
│   └── bezna-magii.jpg
└── files/            ← epub файлы по папкам
    └── bezna-magii/
        ├── Том 1.epub
        └── Том 2.epub
```

---

## Как запустить локально (для проверки)

Нельзя просто открыть index.html двойным кликом — браузер заблокирует загрузку data.json.
Нужен локальный сервер. Самый простой способ:

**Если установлен Python 3:**
```bash
cd ranobe-archive
python3 -m http.server 8000
```
Затем открой http://localhost:8000

**Если установлен Node.js:**
```bash
npx serve ranobe-archive
```

## Как выложить на GitHub Pages (бесплатно, навсегда)

### Шаг 1 — Создай репозиторий

1. Зайди на https://github.com и войди в аккаунт
2. Нажми кнопку **New** (зелёная, сверху слева)
3. Назови репозиторий: `ranobe-archive`
4. Выбери **Public** (обязательно для бесплатного Pages)
5. Нажми **Create repository**

### Шаг 2 — Загрузи файлы

**Способ A: через браузер (проще)**

1. В репозитории нажми **Add file → Upload files**
2. Перетащи все файлы: `index.html`, `admin.html`, `data.json`
3. Нажми **Commit changes**
4. Повтори для папок `covers/` и `files/` (GitHub не умеет загружать пустые папки — положи хотя бы один файл)

**Способ B: через Git (рекомендую)**

```bash
cd ranobe-archive
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/ТВО_ИМЯ/ranobe-archive.git
git push -u origin main
```

### Шаг 3 — Включи GitHub Pages

1. В репозитории перейди в **Settings** (вкладка сверху)
2. В левом меню найди **Pages**
3. В разделе **Branch** выбери `main` и папку `/ (root)`
4. Нажми **Save**
5. Через 1-2 минуты сайт будет доступен по адресу:
   `https://ТВО_ИМЯ.github.io/ranobe-archive/`

## Как добавить новое ранобэ

1. Открой `https://ТВО_ИМЯ.github.io/ranobe-archive/admin.html`
2. Вкладка **«Новое ранобэ»**
3. Заполни форму: название, автор, жанры, обложка, epub файлы
4. Нажми **«Сгенерировать запись»**
5. Скопируй JSON-блок
6. Открой `data.json` в репозитории (на GitHub: кликни на файл → карандаш)
7. Найди `"ranobe": [` и вставь блок сразу после `[`, добавь запятую `,` после блока
8. Загрузи обложку в папку `covers/` с нужным именем (как указано в инструкции)
9. Загрузи epub файлы в папку `files/slug-названия/`
10. Нажми **Commit changes** — сайт обновится через ~30 секунд

## Как добавить том в существующее ранобэ

1. Открой `admin.html`
2. Вкладка **«Добавить файлы»**
3. Загрузи свой `data.json` в форму
4. Выбери нужное ранобэ
5. Добавь epub файлы
6. Скопируй результат и замени старую запись в `data.json`
7. Загрузи epub файлы на GitHub

## Формат data.json

```json
{
  "ranobe": [
    {
      "id": 1,
      "title": "Название ранобэ",
      "author": "Автор",
      "genres": ["Фэнтези", "Исэкай"],
      "cover": "covers/slug.jpg",
      "files": [
        {
          "name": "Том 1.epub",
          "size": "2.4 МБ",
          "url": "files/slug/Том 1.epub"
        }
      ]
    }
  ]
}
```
